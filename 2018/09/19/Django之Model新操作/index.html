Book.objects.all()会返回数据库中所有的图书数据，但是当你要查询作者叫做东野圭吾的全部书籍呢？传统的做法（通常的做法）是这么写 Books.objects.filter(title="东野圭吾")。但是我们也可以利用Django提供的的另外一种方法——Manager.

定义如下：

A Manager is the interface through which database query operations are provided to Django models. At least one Manager exists for every model in a Django application.

它是一个接口，主要的作用是可以理解为是一个桥梁，一个为Model和数据库交互的桥梁。
