win10环境下Opencv3+VS2017环境配置

发表于 2018-08-30
字数统计: 695 | 阅读时长 ≈ 3

最近刚开始入门OpenCV，基础版的安装配置过程可以说是无痛的，但是因为后期学习的需要编译安装成了绕不开的步骤。本文主要介绍两种安装方法，配置部分相似我也会详细描述。新手踩坑真的是一条不归路啊~

一、简易版

之所以说是简易安装版，是因为这种安装方式安装的过程相对简单，对于基本的学习足够。但是很多opencv的模块不会安装，比如后面的目标跟踪模块（tracking）。话不多说，开始~

1.下载和安装

1.1 下载网址

Opencv官网下载（推荐）： https://opencv.org/releases.html

可以根据实际需求下载相关版本，这里我们选择红色框选的。

1.2 安装过程

1.3 安装结束

安装结束，找到刚才目录下发现有一个opencv 的文件夹被创建。打开出现build和sources文件夹（其他文件请忽略）,继续打开build文件夹，是以下文件结构，到此安装步骤完成。

2. 配置阶段

2.1 配置环境变量

对于配置各种环境的小伙伴来说这一定不陌生，即使是第一次配置也没有关系。

介绍一种比较简单直接的方法，通过cmd设置。

第一步 win+R 输入cmd 按回车

第二步 setx PATH "%PATH%;############" 将######部分替换成 opencv中的bin文件夹的路径，例如 setx PATH "%PATH%;D:\Program Files\opencv\build\bin" 就ok啦

2.2 打开vs2017进行配置

因为这里的配置是针对每一个解决方案的，所以我们要创建一个解决方案（也就是一个项目）

创建项目之后，依次打开 视图 –> 其他窗口 –> 属性管理器，出现如下界面

双击这里的项目名称，我这里叫LearnOpenCV，出现以下界面（我只截取了左侧部分图片）

我们来配置这里的两个部分 VC++目录、链接器

2.2.1 VC++目录

这里面配置如图两个内容

包含目录添加 (次序不限)

  • [opencv安装目录]\build\include
  • [opencv安装目录]\build\include\opencv

库目录添加(x86或者x64根据安装目录下的具体情况决定)

  • [opencv安装目录]\build\x86\vc15\lib
